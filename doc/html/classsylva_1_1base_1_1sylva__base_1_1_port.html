Detailed Description

Class for port.

One Port object will be used to represent one data communication port on an actor in SDF graph, where a port on an actor emits or receives one data token per clock cycle.

The port is either input or output. The IO direction (input or output) is not defined in this class.

  • One Port is an input port if it is placed in the Actor input_ports
  • One Port is an input port if it is placed in the Actor output_ports
