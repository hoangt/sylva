Detailed Description

Class for communication edge.

One Edge object will be used to represent one data communication edge between two Port objects of two Actor objects in SDFG, where a port on an actor emits or receives a constat number of data token per clock cycle. Each Edge object has one source actor, one source port, one destination actor and one destination port.
