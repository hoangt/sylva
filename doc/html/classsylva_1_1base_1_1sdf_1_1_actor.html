Detailed Description

Class for SDF actor.

["SDFG"]
The name of the Actor is its function name, e.g a, b. The index of the Actor is the unique integer identifier of the actor in an SDFG or HSDFG. All the data source actors of one Actor object will have lower indexes than that Actor object. e.g a.index = 0, d.index = 1, b.index = 2, c.index = 3
