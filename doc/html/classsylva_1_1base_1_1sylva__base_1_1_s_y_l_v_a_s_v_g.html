Parameters
clsThe cls
local_kwargsThe local kwargs This is the keyword arguments passed from the caller method. The name of the svg element, e.g rect or text, is extracted from the caller method, e.g add_rect or add_text.
Returns
One SYLVASVG object The parent of the generated object will always be the self object. E.g invoking self.add_rect with label argument will call SYLVASVG.generate() to create one rect svg element and one text svg element. The parent of both generted SYLVASVG objects is self.
